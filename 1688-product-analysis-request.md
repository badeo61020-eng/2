# 1688 商品分析依頼

## 対象URL

- https://detail.1688.com/offer/1001596981752.html

## 依頼内容

- 商品ページを分析する
- 写真、素材、サイズ、仕様、訴求ポイントを確認する
- EC向けに最適なLPの方向性を判断する
- 必要に応じて Amazon 向き / 楽天向きの見せ方も判断する
- 最終的に PASBECONA 形式でLPを作成する

## 現時点の制約

- この環境では Chrome アプリを直接操作して商品ページを開くことはできない
- そのため、正確な分析にはユーザーから商品情報の共有が必要

## 必要な共有情報

- メイン画像
- 商品タイトル
- 素材説明
- サイズ情報
- 詳細画像
- 商品説明文
- レビュー部分の画像またはテキスト

## 受領後に返す内容

- 商品分析
- 写真ベースの訴求軸
- EC向けの見せ方
- 想定される不安点
- PASBECONA形式のLP
