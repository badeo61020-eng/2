# カーテンレールの色をエンドキャップに合わせるプロンプト

## 用途
カーテンレールの色だけがエンドキャップと合っていないときに、レール本体の色味をエンドキャップにそろえて自然な一体感を出すための Nano Banana Pro 用プロンプトです。

## メインプロンプト
```text
この画像を補正して、カーテンレール本体の色をエンドキャップの色に合わせてください。エンドキャップの白い色味、明るさ、質感にそろえて、レール全体も同系統の自然な白に調整してください。黄味やグレー感が残らないようにしつつ、真っ白に塗りつぶした不自然な見え方ではなく、素材感と立体感が残る自然な仕上がりにしてください。

レールの形状、太さ、カーブ、端部の構造、エンドキャップの形状は変更しないでください。色の調整はレール本体のみに行い、エンドキャップと違和感なく一体化して見えるようにしてください。接合部で色差が出ないようにし、左右とも均一な見え方に整えてください。

壁、窓、カーテン、光、構図は変更せず、編集はカーテンレールの色味調整のみに限定してください。切り抜き感、塗りつぶし感、質感消失のない、自然な実写の商品写真として仕上げてください。
```

## 補強用の一文
```text
エンドキャップだけが白く、レールだけが別色に見えないように、同じ素材の一体パーツのような自然な色合わせにしてください。
```
