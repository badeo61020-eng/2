# カーテンレール端部にエンドキャップを追加する日本語プロンプト

## 用途

図1のカーテンレール右端の開口部に、図2・図3を参考にした同一形状の `エンドキャップ` を自然に追加するための画像編集プロンプトです。  
レール本体の形状、曲線、材質感、背景、構図は変えず、右端だけを自然に補完したいときに使います。

## メインプロンプト

```text
図1をベースに、部分的な写実編集を行ってください。カーテンレール全体の形状、曲線、金属の質感、色、背景、構図はそのまま維持してください。

右側の開口している端部にのみ、参考画像の白っぽい樹脂製エンドキャップを自然に取り付けてください。追加するのは右端の端部パーツだけで、レール本体のデザインや他の部分は変更しないでください。

エンドキャップは、図2と図3を同一部品の別角度参考として用い、実物らしい三次元形状で再現してください。特に以下を正確に反映してください。

- レール断面にぴったり収まる差し込み形状
- 側面から見た厚み、段差、ふくらみ、立体感
- 下部の小さな金属リングの位置関係
- レールとの接合部の自然な陰影、接触、遮蔽関係

最も重要なのは、図1右端の開口断面のパースと向きに正確に合わせることです。参考画像の正面形状をそのまま無理に引き伸ばしたり、平面的に貼り付けたりせず、図1の視点に従って自然な角度で装着してください。

端部パーツのサイズはレール断面と整合する実寸感にし、浮き、ズレ、変形、過度な誇張を避けてください。光の向き、影、シャープさ、反射、色味も元画像と一致させ、最初からその製品写真に付いていたような自然な仕上がりにしてください。

商品画像として、清潔感のある高品質な写実表現にしてください。右端以外は編集しないでください。
```

## 補強用の一文

角度が合わない場合は追加:

```text
参考画像は同一エンドキャップの形状把握のためだけに使用し、生成時は必ず図1右端開口部の現在のパースと向きに従ってください。参考画像の視点をそのまま貼り付けないでください。
```

レール全体を勝手に変えられる場合は追加:

```text
右端の開口部にエンドキャップを追加する以外の変更は行わず、レール全体の形状や構図は元画像のまま維持してください。
```

端部パーツが大きくなりすぎる場合は追加:

```text
エンドキャップはレール断面とつり合う控えめで実在感のあるサイズにし、誇張した展示パーツのようにしないでください。
```
