# カーテンレール端部にエンドキャップを追加する編集提示词

## 用途

用于基于原图进行局部真实感编辑，在不改变弯曲カーテンレール主体形状、构图、背景和材质表现的前提下，只给开口端自然加上参考图同款 `エンドキャップ`。

适合用于支持「图像编辑 / inpaint / reference edit」的模型。  
如果模型支持多参考图，请将图1作为底图，图2与图3作为同一零件的参考角度图一起输入。

## 主提示词

```text
请基于图1进行局部真实感编辑，保留整条弯曲カーテンレール的整体形状、弧线走势、金属材质、颜色、背景和构图不变。

只在右侧已经开口的端部增加一个与参考图一致的白色偏象牙色树脂エンドキャップ，让它像原本就安装在这根レール上一样自然。

新增的エンドキャップ必须严格匹配右端开口断面的透视角度与安装方向，不要把参考零件的正面外形强行拉伸成错误角度。请根据参考图2和图3综合还原这个零件的真实三维结构，包括：
- 与レール断面贴合的卡入式主体轮廓
- 合理的厚度、转折、台阶和侧面体积
- 下部小金属环的位置关系
- 零件与レール连接处真实的遮挡、接缝、阴影和受光

让这个エンドキャップ准确安装在右端开口处，尺寸协调，连接牢靠，比例真实，不漂浮、不歪斜、不悬空。
整体呈现保持为干净、写实、商品摄影级别的产品图。不要改变レール本体结构，不要额外增加配件，不要修改左侧弯曲部分，不要重画整张图。

重点要求：
- 只编辑右端端部区域
- 端盖与开口方向完全一致
- 保持零件真实厚度，不要压扁，不要拉长，不要变形
- 光线、清晰度、阴影、反射与原图统一
- 避免贴图感、拼接感、边缘发白、轮廓光晕
```

## 负面提示词

适用于支持 negative prompt 的模型。

```text
deformed end cap, distorted plastic part, stretched part, flattened part, wrong perspective,
wrong angle, wrong scale, floating cap, detached cap, pasted object, cutout look,
halo edge, mismatch lighting, mismatch shadow, mismatch sharpness,
changed rail shape, changed rail opening, altered product design, extra accessories,
duplicate metal ring, oversized ring, cartoon, illustration, blur, low detail
```

## 短版补强句

当模型容易把零件做错角度时，可额外补一句：

```text
参考图2和图3只用于还原同一个エンドキャップ的真实三维形状，生成时必须服从图1右端开口的当前透视角度，不能把参考图视角直接硬贴上去。
```

如果模型容易乱改整条レール，可额外补一句：

```text
除右端开口处新增端盖外，其他区域保持原图不变，不重绘整条カーテンレール。
```

如果模型容易把零件做得太大或太抢眼，可额外补一句：

```text
端盖尺寸应与レール断面严格匹配，作为真实安装零件自然收尾，而不是夸张展示件。
```

## 使用建议

- 优先使用局部编辑或参考编辑模式，不建议直接纯文生图。
- 如果工具支持遮罩，只遮住图1右端开口区域，让模型只在该区域补件。
- 如果工具支持多参考图，将图2、图3同时作为同一零件的不同角度参考输入。
- 如果金属环不是必须展示，可在提示词里追加一句：`金属环仅小幅自然露出，不要夸张突出。`
