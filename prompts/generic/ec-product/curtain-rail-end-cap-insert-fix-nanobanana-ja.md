# カーテンレール端部エンドキャップ嵌め込み修正プロンプト（Nano Banana Pro用）

## 用途

エンドキャップの向きは合っているが、サイズが小さい、レール端部に十分入り込んでいない、先端に被さっているだけに見える場合の修正用プロンプトです。  
`カーテンレール端部の開口断面の内側に、適切なサイズでしっかり差し込まれている状態` に補正したいときに使います。

## 指示文

```text
現在のエンドキャップは方向は正しいので、その向きは維持したまま、サイズと取り付き方だけを修正してください。

エンドキャップを少し大きくして、カーテンレール右端の開口部にしっかり差し込まれ、断面の内側にきちんと嵌め込まれている状態にしてください。
レールの先端に小さく付いている見え方ではなく、レール端部の断面に合った寸法で、内部に収まりながら固定されている自然な取り付け状態に修正してください。

特に、エンドキャップの差し込み部分がレールの内側に入っていること、接合部に隙間がないこと、レール断面とサイズが合っていることを重視してください。
浮いて見える状態、小さすぎる状態、先端にただ被せただけの状態は避けてください。

エンドキャップ本体の厚み、奥行き、段差、下部リングの位置関係は自然に保ちつつ、レールとの接続部分に実在感のある接触、陰影、奥行きをつけてください。

カーテンレール本体の形状、曲線、金属の質感、色、背景、構図は変更しないでください。
修正対象は右端のエンドキャップ周辺のみです。
商品写真として、最初からこのエンドキャップが正しく装着されていたような自然で写実的な仕上がりにしてください。
```

## 補強用の一文

```text
エンドキャップはレール端部の外側に付けるのではなく、開口断面の内側に差し込まれて固定されている見え方にしてください。
```
