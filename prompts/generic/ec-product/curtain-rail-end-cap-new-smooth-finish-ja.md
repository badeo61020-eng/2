# エンドキャップを新品できれいに見せる光滑仕上げプロンプト

## 用途
エンドキャップ単品画像に傷、擦れ、表面の荒れ、くすみが見えるときに、新品らしい清潔感と滑らかな表面に整えたいときの Nano Banana Pro 用プロンプトです。

## メインプロンプト
```text
この画像のエンドキャップを、新品のようにきれいで滑らかな状態に補正してください。表面に見える細かな傷、擦れ、くすみ、ムラ、ざらつき感を自然に抑え、清潔感のあるなめらかな白系樹脂パーツに見えるようにしてください。

形状、輪郭、寸法感、厚み、穴位置、金具の構造や位置は変更しないでください。エンドキャップそのもののデザインは保ったまま、表面品質だけを改善してください。古びた感じ、中古感、使用感はなくし、新品の商品撮影のような整った見え方にしてください。

白い樹脂部分は、マットすぎず、不自然なテカリすぎでもない、上品で均一な表面に整えてください。素材感と立体感は残しつつ、表面がなめらかで清潔に見えるようにしてください。金具部分も汚れやくすみが強ければ自然に整え、全体として新品らしい高品質な商品写真にしてください。

背景は白を基調に清潔に見せ、構図、角度、パーツの向きは変えないでください。文字、寸法線、余計な装飾、別パーツの追加は行わず、単品の商品画像として自然で美しい仕上がりにしてください。
```

## 補強用の一文
```text
中古品や使用済み部品のような荒れた表面ではなく、出荷前の新品パーツのように、表面が均一でなめらかに整った状態へ補正してください。
```
