# カーテンレール端部エンドキャップ差し替えプロンプト（Nano Banana Pro用）

## 用途

カーテンレール端部に付いている既存のエンドキャップを、参考画像の別形状エンドキャップへ差し替えるためのプロンプトです。  
図2の背面形状と図3の側面形状をあわせて参照し、同一部品として自然に再構成しながら、レール端部へ正しく装着された見え方にしたいときに使います。

## メインプロンプト

```text
カーテンレール端部に付いている現在のエンドキャップを、参考画像のエンドキャップ形状に差し替えてください。

図2は背面側の形状参考、図3は側面側の形状参考として使い、同一のエンドキャップとして一貫した三次元形状で再現してください。
特に、背面から見た外形、側面から見た厚みと段差、下部リングの位置関係を参考画像に合わせてください。

エンドキャップはカーテンレール端部の開口断面に正しく合うサイズで、開口部の内側へ自然に差し込まれ、端部にしっかり装着された状態にしてください。
端部の外側に貼り付いている見え方ではなく、レール断面に沿って内部に収まり、接合部に隙間のない一体化した見え方にしてください。

図2の背面形状を優先して、正面側の外形や凹凸の印象を合わせてください。
同時に図3の側面形状を反映し、側面から見た厚み、奥行き、段差、下部リングの付き方が自然になるようにしてください。
参考画像をそのまま平面的に貼るのではなく、レール端部の現在のパースに従って自然な角度で装着してください。

カーテンレール本体の形状、曲線、材質、色、背景、構図は変えないでください。
修正対象は端部のエンドキャップ部分のみです。
商品写真として自然で、最初からその部品が正しく装着されていたような写実的な仕上がりにしてください。
```

## 補強用の一文

```text
図2の背面形状と図3の側面形状を統合した同一部品として再現し、カーテンレール端部の開口断面にぴったり合うサイズで自然に嵌合させてください。
```
