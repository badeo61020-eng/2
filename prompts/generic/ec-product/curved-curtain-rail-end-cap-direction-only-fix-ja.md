# 曲がりカーテンレール端部エンドキャップ向きのみ修正プロンプト

## 用途

室内設置された曲がりカーテンレール画像で、端部エンドキャップの向きだけを参考画像の方向に合わせて修正したい場合のプロンプトです。  
サイズ、位置、室内背景、レール本体、カーテンはそのまま維持し、`向きだけ` を直したいときに使います。

## メインプロンプト

```text
窓まわりの曲がりカーテンレール端部について、エンドキャップの向きだけを図2の方向に合わせて修正してください。

修正対象は端部のエンドキャップの向きのみです。
図2のエンドキャップの向きを参考にして、端部に付いている現在のエンドキャップを同じ向きに直してください。

サイズ、位置、取り付け位置、レールへの嵌合状態はそのまま維持し、向き以外は変更しないでください。
エンドキャップの形状自体を別物に変えたり、大きさを変えたり、位置をずらしたりしないでください。

カーテンレール本体の形状、曲線、材質、色、ランナー、カーテン、壁、天井、窓、背景は一切変更しないでください。
端部エンドキャップの前後方向だけを自然に回転・修正し、最初からその向きで正しく付いていたように見せてください。

貼り付け感、浮き、不自然な影、形状の崩れ、サイズ変化は避けてください。
商品写真として自然で、違和感のない写実的な仕上がりにしてください。
```

## 補強用の一文

```text
変更するのはエンドキャップの向きだけで、サイズ・位置・レール本体・カーテン・室内背景は絶対に変えないでください。
```
