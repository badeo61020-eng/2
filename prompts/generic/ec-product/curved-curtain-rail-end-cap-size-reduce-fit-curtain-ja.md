# 曲がりカーテンレール端部エンドキャップ縮小補正プロンプト

## 用途

室内設置された曲がりカーテンレール画像で、左右両端のエンドキャップがカーテンに対して大きく見えすぎる場合に、カーテンのサイズ感に合わせて自然な大きさへ補正するためのプロンプトです。  
エンドキャップだけを少し小さくし、全体のバランスを自然に整えたいときに使います。

## メインプロンプト

```text
この室内画像では、左右両端のエンドキャップがカーテンのサイズ感に対してやや大きく見えるため、エンドキャップだけを少し小さくして自然なバランスに修正してください。

左右のエンドキャップは、現在の形状、向き、取り付け位置、レール端部との接続状態はできるだけ維持したまま、サイズだけを控えめに縮小してください。
カーテンのボリューム、ひだの幅、全体の見え方に対して違和感のない寸法にし、端部だけが不自然に目立たないようにしてください。

重要なのは、エンドキャップを小さくしても、レール端部に正しく取り付けられている自然な見え方を保つことです。
小さくしすぎたり、細くしすぎたり、形が変わったり、位置がずれたりしないでください。
左右で大きさがばらつかないようにし、両端とも同じ部品として整合したサイズ感にしてください。

カーテンレール本体の形状、曲線、材質、色、カーテン、窓、壁、天井、床、背景は変更しないでください。
修正対象は左右両端のエンドキャップのサイズ感のみです。

商品写真として自然で、最初からこのサイズで付いていたような写実的な仕上がりにしてください。
```

## 補強用の一文

```text
左右両端のエンドキャップだけを、カーテンのスケール感に合うよう少し小さくし、形状・向き・位置・レール本体・カーテンは変えないでください。
```
