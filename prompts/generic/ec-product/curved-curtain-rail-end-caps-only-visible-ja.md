# 曲がりカーテンレール端部のみ見せる修正プロンプト

## 用途

室内設置された曲がりカーテンレール画像で、左右両端のエンドキャップだけを見せ、中間部分のカーテンレール本体はカーテンの後ろに隠れて見えない状態にしたい場合のプロンプトです。  
端部パーツの存在は見せつつ、レール本体の露出を減らして、より自然な設置イメージにしたいときに使います。

## メインプロンプト

```text
この室内画像では、左右両端のエンドキャップだけが見える状態に修正してください。

左右の端部にあるエンドキャップはそのまま自然に見せてください。
一方で、中間部分を含むカーテンレール本体はカーテンの後ろに隠し、正面からはほとんど見えない状態にしてください。

見せたいのは両端のエンドキャップだけで、その他のレール金属部分やレール本体の見える量は大きく減らしてください。
カーテンが自然に前に来て、レール本体を隠している見え方にしてください。

カーテンの吊り位置、ひだ、落ち感、ランナーとのつながりは自然に保ち、無理に切り貼りしたような見え方にしないでください。
カーテンがレール本体を隠していても、実際にその後ろにレールが通っていると分かる自然な納まりにしてください。

左右のエンドキャップのサイズ、位置、向き、取り付け状態は自然に保ってください。
カーテンレール本体の形状を別物に変えたり、エンドキャップを消したり、窓・壁・天井・床・背景を大きく変えたりしないでください。

商品写真として自然で、後から隠したような不自然さのない、写実的な仕上がりにしてください。
```

## 補強用の一文

```text
見えるのは左右両端のエンドキャップだけにし、中間のカーテンレール本体はカーテンの後ろに隠れて正面から見えないようにしてください。
```
