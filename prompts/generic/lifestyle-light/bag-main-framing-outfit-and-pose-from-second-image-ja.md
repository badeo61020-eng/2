# 1枚目のバッグを主役にし、服装とポーズを2枚目に寄せる日本語プロンプト

## 用途

1枚目の画像のバッグを主役として見せつつ、人物の服装とポーズは2枚目の画像の雰囲気に合わせた画像を生成するためのプロンプトです。  
バッグの商品訴求を優先しながら、人物の見せ方だけを2枚目寄りに調整したいときに使います。

## メインプロンプト

```text
1枚目の画像のバッグを主役にした、高品質な実写ライフスタイル写真を生成してください。

バッグ本体のデザイン、形状、サイズ感、色味、素材感、ディテールは1枚目の画像を基準に忠実に反映してください。商品としての見え方を最優先し、バッグが最も目立つ画角にしてください。

人物の服装とポーズは2枚目の画像を参考にしてください。2枚目のコーディネートの系統、シルエット、袖やトップスの見え方、全体のスタイリングの印象、立ち姿や腕の位置、体の向き、歩き方や構え方を自然に取り入れてください。

ただし、最優先はバッグ訴求です。構図はバッグが画面内で十分大きく、視線の中心になるようにしてください。バッグの全体像、ストラップ、持ち方、体との位置関係が分かりやすいフレーミングにしてください。

人物はバッグを自然に身につけている状態にし、ストラップの掛かり方、肩や腕との接触、バッグの重さによる位置関係、服とのなじみ方を自然に表現してください。バッグを無理に貼り付けたような不自然さは避けてください。

背景はライフスタイル感のある自然な屋外または街中の雰囲気で構いませんが、背景よりもバッグが主役に見えるようにしてください。背景は適度に整理され、やわらかいボケ感があり、商品が埋もれないようにしてください。

全体として、ECやSNSの商品訴求に使える自然で上品な実写写真にしてください。人物はバッグを引き立てる役割にとどめ、服装とポーズは2枚目の印象を反映しつつ、最終的な画作りはバッグ中心でまとめてください。
```

## 補強用の一文

バッグをさらに大きく見せたい場合は追加:

```text
バッグが画面の主役として十分な面積を占めるようにし、上半身中心の画角で商品ディテールが見やすい構図にしてください。
```

2枚目のポーズ再現度を上げたい場合は追加:

```text
人物の体のひねり、腕の位置、肩の角度、歩行感または立ち姿の重心は2枚目のポーズを優先して反映してください。
```

2枚目の服装反映を強めたい場合は追加:

```text
人物のトップス、ボトムス、袖の印象、全体のシルエットは2枚目の服装に寄せてください。ただしバッグが埋もれないようにしてください。
```
