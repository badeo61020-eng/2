# 1枚目のバッグを2枚目の人物に持たせ、2枚目の背景を維持する日本語プロンプト

## 用途

1枚目の画像に写っているバッグだけを参照し、2枚目の画像の人物に自然に持たせるための画像編集プロンプトです。  
2枚目の人物の顔・髪型・体型・服装・ポーズ・構図・背景はできるだけ維持し、バッグだけを自然に差し替えたいときに使います。

## メインプロンプト

```text
2枚目の画像をベースに、部分的な写実編集を行ってください。人物、顔、髪型、体型、服装、ポーズ、背景、構図、カメラ角度、遠近感、光の向きは2枚目のまま維持してください。

1枚目の画像のバッグのみを参照し、そのバッグと同一デザイン・同一形状・同一色味・同一素材感に見えるように、2枚目の人物が自然に持っている状態へ置き換えてください。

最優先は、2枚目の人物写真として自然に成立させることです。単純にバッグを貼り付けるのではなく、2枚目の人物の手の位置、腕の角度、肩の向き、体との接触関係に合わせて、自然な持ち方になるよう再構成してください。

バッグの大きさは人物との対比で自然な実寸感に合わせ、過度に大きくしたり小さくしたりしないでください。持ち手やストラップがある場合は、重力、たわみ、ねじれ、肩や手との接触、衣服への食い込みや隠れ方まで自然に表現してください。

バッグの輪郭に切り抜き感、白フチ、ハロ、浮き、影の不一致が出ないようにしてください。光、影、反射、シャープさ、色温度、質感を2枚目の環境に自然に一致させ、最初からその場で撮影された写真のような仕上がりにしてください。

背景は2枚目の画像の背景をそのまま維持してください。背景の場所、壁、床、家具、景色、ボケ感、明るさ、色味は変更しないでください。バッグを持たせること以外の不要な改変は行わないでください。

全体として、高品質な実写写真として自然で違和感のない仕上がりにしてください。
```

## ネガティブプロンプト

```text
pasted bag, cutout look, floating bag, halo edge, white outline, mismatched lighting,
mismatched shadow, mismatched perspective, wrong hand position, broken fingers,
deformed bag, distorted strap, wrong scale, oversized bag, undersized bag,
extra bag, duplicated bag, changed person, changed face, changed outfit,
changed pose, changed background, illustration, cartoon, low detail, blur
```

## 補強用の一文

バッグの同一性をさらに強めたい場合は追加:

```text
バッグの形状、フラップ、金具、持ち手、ストラップ、縫い目、素材感は1枚目のバッグを忠実に反映してください。
```

人物や背景を勝手に変えられる場合は追加:

```text
編集対象はバッグ周辺だけに限定し、2枚目の人物と背景の見え方はできるだけ保持してください。
```

持ち方が不自然になる場合は追加:

```text
人物の手指、手首、腕、肩の関係を自然に補正し、バッグを無理なく持っている姿勢にしてください。
```
