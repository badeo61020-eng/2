# コーナーカーテンレール写真をプリンセス風に寄せるプロンプト

## 用途
コーナーに曲がったカーテンレールと窓まわりの設置写真を、上品でやわらかなプリンセス風インテリアへ寄せたいときの Nano Banana Pro 用プロンプトです。レールや窓の基本構図は保ちつつ、カーテンと背景全体の雰囲気を姫系に整える用途を想定しています。

## メインプロンプト
```text
この画像を、上品でやわらかなプリンセス風インテリアの雰囲気に編集してください。コーナーに沿ったカーテンレールと窓の基本構図はそのまま保ち、空間全体をロマンティックで可憐な印象に整えてください。白を基調に、アイボリー、淡いピンクベージュ、やさしいパール感を感じる高級感のある色調にしてください。

カーテンはよりプリンセス風に見えるよう、軽やかな透け感、繊細なドレープ、ふんわりした柔らかさ、上品でエレガントな布の質感に整えてください。必要に応じて、控えめなフリル感や柔らかな布の重なりを自然に加え、姫系らしい繊細さを出してください。ただし、商品写真として不自然な過剰装飾にはせず、実在する上品な室内として成立する範囲にとどめてください。

背景は、今のシンプルな壁と窓まわりをやわらげ、プリンセス風の室内に自然になじむ内装へ整えてください。壁は明るく上品な雰囲気にし、必要なら繊細なモールディング、淡い装飾、やわらかな陰影を加えてください。窓フレームまわりや光の印象も、ふんわり明るく、清潔感と夢のある見え方に整えてください。空間全体は甘すぎず、大人向けの上質なプリンセスルームの世界観にしてください。

カーテンレールの形状、位置、曲がり方、窓の位置、撮影アングルは大きく崩さないでください。実写のインテリア写真として自然で、美しく、上品な仕上がりにしてください。
```

## 補強用の一文
```text
安っぽい派手な姫系ではなく、白と淡いピンクベージュを基調にした、上品で高級感のあるプリンセス風インテリアとしてまとめてください。
```
