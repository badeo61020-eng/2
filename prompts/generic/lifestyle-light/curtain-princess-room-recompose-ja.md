# カーテンと背景を上品なプリンセス風に寄せるプロンプト

## 用途
現在のカーテン設置写真を、甘すぎず上品なプリンセス風インテリアへ寄せたいときの Nano Banana Pro 用プロンプトです。カーテンを主役に保ちながら、背景全体の雰囲気を姫系・ロマンティック系に整える用途を想定しています。

## メインプロンプト
```text
この画像を、上品で柔らかなプリンセス風インテリアの雰囲気に編集してください。カーテンは主役として見せたまま、全体の空間をロマンティックで可憐な印象に整えてください。白を基調に、やわらかなアイボリー、淡いピンクベージュ、繊細なパール感を感じる高級感のある空間にしてください。

カーテンはよりプリンセス風に見えるよう、上品なドレープ感、ふんわりした軽やかさ、やさしい透け感、繊細でエレガントな質感に整えてください。必要に応じて、フリル感や柔らかな布の重なり、上質な姫系テキスタイルらしい雰囲気を自然に加えてください。ただし、商品として不自然な過剰装飾にはせず、実在する上品な室内写真として成立する範囲にとどめてください。

背景は、今の無機質な壁と床の印象をやわらげ、プリンセス風の室内に自然になじむ空間へ変更してください。壁はやさしい明るさのある上品な内装にし、必要なら繊細なモールディング、淡い装飾、エレガントな陰影を加えてください。窓まわり、床、周辺の小物も、全体に調和するロマンティックで上質な見え方に整えてください。空間全体は明るく、ふんわりした自然光が入る、清潔感と夢のある雰囲気にしてください。

全体として、子どもっぽい玩具風や派手すぎる装飾にはせず、大人向けの上品なプリンセスルームの世界観にしてください。カーテンレール、カーテンの基本構図、窓位置、撮影アングルは大きく崩さず、実写のインテリア写真として自然で美しい仕上がりにしてください。
```

## 補強用の一文
```text
甘すぎる安っぽい姫系ではなく、白と淡いピンクベージュを基調にした上品で高級感のあるプリンセス風インテリアとしてまとめてください。
```
