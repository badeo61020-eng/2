# この画像の人物が正面を向いている構図にする日本語プロンプト

## 用途

参照画像の人物、服装、バッグ、背景の雰囲気を保ちながら、人物がカメラに対して自然に正面を向いている画像を生成するためのプロンプトです。  
商品としてのバッグ訴求を維持しつつ、人物の向きだけを正面寄りに整えたいときに使います。

## メインプロンプト

```text
参照画像をベースに、自然な実写ライフスタイル写真として再構成してください。

人物はカメラに対して正面を向いている構図にしてください。体の向き、肩の開き、首の向き、胸元の見え方を自然に正面寄りへ調整し、不自然なねじれや無理な姿勢にならないようにしてください。

服装は参照画像のまま維持してください。白のリブニットトップス、黒の縁取りフリル、グレー系ボトムス、アクセサリーの印象は変えないでください。

バッグも参照画像のまま維持してください。黒いクリアショルダーバッグ本体、透明フロント、内部の3体のぬいぐるみ、ストラップ、金具、サイズ感、体に対する位置関係を忠実に保ってください。

バッグは商品として自然に見えるように、体の正面向きに合わせてストラップの掛かり方、バッグの正面の見え方、体との接触、重さによる位置関係を自然に調整してください。バッグを貼り付けたような不自然さは避けてください。

背景は参照画像と同じく、明るい屋外の歩道とやわらかくボケた緑のある雰囲気を維持してください。光の向き、明るさ、色温度、空気感も元画像に合わせてください。

全体として、自然で上品な実写写真にしてください。人物が正面を向いていても、商品バッグが見やすく、ECやSNSで使いやすい仕上がりにしてください。
```

## 補強用の一文

顔の向きをさらに正面寄りにしたい場合は追加:

```text
顔、首、肩はカメラに対して自然に正対するようにしてください。
```

バッグの見え方を優先したい場合は追加:

```text
人物が正面を向いていても、バッグ正面がしっかり見え、ぬいぐるみ3体が分かりやすく見える角度にしてください。
```

立ち姿の自然さを強めたい場合は追加:

```text
重心、肩の高さ、腕の位置、ストラップの張り方が自然な立ち姿になるように調整してください。
```
