# 1枚目のポーチの中身を2枚目の内容に差し替える日本語プロンプト

## 用途

1枚目の画像に写っているポーチ本体、形状、色、素材感、置かれ方は維持しつつ、ポーチの中に入っている物だけを2枚目の画像の内容へ自然に差し替えるための画像編集プロンプトです。  
ポーチ外観や周囲の背景は変えず、中身だけを違和感なく入れ替えたいときに使います。

## メインプロンプト

```text
1枚目の画像をベースに、部分的な写実編集を行ってください。ポーチ本体の形状、サイズ、色、素材感、ファスナー、縫製、開き方、置かれている位置、背景、構図、カメラ角度、光の向きは1枚目のまま維持してください。

編集対象はポーチの中身のみです。2枚目の画像に写っているアイテムの種類、見た目、色味、素材感、数量感、配置の印象を参考にして、1枚目のポーチの中身を自然に入れ替えてください。

重要なのは、1枚目のポーチ写真として自然に成立させることです。単純に物を貼り付けるのではなく、ポーチの開口部の大きさ、深さ、見込み角度、収納量に合わせて、中身が実際に入っているように再構成してください。

中身はポーチ内部の空間に収まる自然なサイズ感にしてください。物が大きすぎて不自然にはみ出したり、小さすぎて不自然に見えたりしないようにしてください。必要に応じて、重なり、隠れ、奥行き、接触、影を自然に表現してください。

ポーチの内側と中身の接触部分には自然な陰影と遮蔽を付け、切り抜き感、白フチ、ハロ、浮き、不一致な影が出ないようにしてください。光、影、反射、シャープさ、色温度を1枚目の撮影環境に一致させ、最初からそのポーチの中に入っていたような自然な仕上がりにしてください。

ポーチ本体の外観や背景は変更しないでください。中身の入れ替え以外の不要な改変は行わないでください。

全体として、高品質な実写写真として自然で違和感のない仕上がりにしてください。
```

## 補強用の一文

2枚目の内容の再現度をさらに上げたい場合は追加:

```text
2枚目のアイテム構成、色味、素材感、見える範囲の特徴をできるだけ忠実に反映してください。
```

ポーチ本体まで勝手に変えられる場合は追加:

```text
ポーチ本体の外形、開口部、ファスナー、布地、背景には手を加えず、中身だけを編集してください。
```

中身の収まりが不自然になる場合は追加:

```text
ポーチ内部の容積と開口方向に合わせて、中身の重なり方、奥行き、見え方を現実的に調整してください。
```
