# ぬいポーチのストラップ部分を接写し、似た系統で服装だけ変える日本語プロンプト

## 用途

参照画像のぬいポーチとストラップ部分を主役にした接写画像を生成するためのプロンプトです。  
全体の雰囲気、ナチュラルな屋外スナップ感、女性らしいやわらかい印象は保ちつつ、服装だけは似た系統で別コーデに変えたいときに使います。

## メインプロンプト

```text
参照画像の黒いクリア素材のぬいポーチと、そのショルダーストラップ部分に視線が集まるような、自然な接写の実写画像を生成してください。

主役はストラップの取り付け部分、ストラップの質感、金具、肩から脇にかけての掛かり方です。バッグ本体も一部見せて構いませんが、構図はストラップまわりのディテールが最も分かりやすい近接フレーミングにしてください。

ポーチ本体は参照画像と同じ系統にしてください。黒系の本体、透明フロント、ぬいぐるみが見える仕様、細めのショルダーストラップ、金具付きのデザインを維持してください。

人物は20代前半の女性。参照画像と同じく、やわらかく上品で日常スナップ風の雰囲気にしてください。背景は明るい屋外の歩道や街路樹のある場所で、自然光の入るやさしいボケ背景にしてください。

服装は参照画像と似た系統のフェミニンカジュアルにしつつ、同一衣装にはしないでください。たとえば、淡色のリブニットややわらかい質感のトップスを使い、全体のテイストは近いまま、袖デザイン、配色、ボトムスを変えて別コーデにしてください。

服装変更の目的は印象を少し変えることだけで、商品の見え方を邪魔しないでください。ストラップが肩に自然に沿い、服に無理なく接し、金具や縫製や素材感が分かるようにしてください。

不自然な切り抜き感、浮き、破綻した肩や腕、歪んだストラップ、壊れた金具は避け、ECやSNS商品訴求に使える高品質な実写写真にしてください。
```

## 補強用の一文

ストラップ金具をより見せたい場合は追加:

```text
ストラップの接続金具、縫製、素材の厚み、肩へのかかり方がはっきり分かる角度にしてください。
```

服装差分をもう少し明確にしたい場合は追加:

```text
参照画像と同じ白フリルニットは避け、近い上品カジュアル路線の別トップスと別ボトムスに変更してください。
```

接写感をさらに強めたい場合は追加:

```text
胸元から腰上あたりまでのフレーミングを基本にし、ストラップ周辺の情報量を優先してください。
```
