# Image Request Terms (JA)

## 1. Look / Mood

- `高級感`
  - 言い換え: 上質、洗練、低彩度、上品な陰影、深みのある反射、不要なテカリなし
  - 使い方: `全体は低彩度で、上品な陰影と深みのある反射を持つ高級感のあるトーンにする`

- `おしゃれ`
  - 言い換え: 洗練、都会的、ミニマル、整った配色、余計な情報が少ない
  - 使い方: `都会的でミニマルなスタイリングに整える`

- `自然`
  - 言い換え: 不自然な演出なし、現実的、過剰補正なし、無理のない人体・光・遠近感
  - 使い方: `現実的な人体バランスと自然な陰影で仕上げる`

- `プロっぽい`
  - 言い換え: 商品撮影ライティング、整理された構図、意図的な余白、主役が明確
  - 使い方: `商品撮影ライティングで主役が明確に見える構図にする`

## 2. Composition

- `構図`
  - 言い換え: 画角、カメラ距離、被写体サイズ、被写体位置、余白、俯瞰/水平/斜め

- `余白を作る`
  - 言い換え: 文字配置用の空間、上部余白、左右余白、主役をやや下げる
  - 使い方: `上部中央にコピー配置用の余白を確保する`

- `引きで見せる`
  - 言い換え: カメラを後退、中景、全体の関係性を見せる

- `寄りで見せる`
  - 言い換え: クローズアップ、ディテール重視、主役の占有率を上げる

## 3. Lighting / Tone

- `陰影をつける`
  - 言い換え: 明暗グラデーション、面の立体感、エッジに沿う細いハイライト
  - 使い方: `正面の広い面に明暗グラデーションを入れて立体感を出す`

- `黒つぶれを防ぐ`
  - 言い換え: 弱い補助光、階調保持、暗部の情報を残す
  - 使い方: `弱い補助光で黒つぶれを防ぎ、黒の階調を残す`

- `テカリを抑える`
  - 言い換え: 不要な白飛び反射を抑える、鏡面反射を弱める

- `低彩度`
  - 言い換え: 彩度控えめ、落ち着いた色調、広告向けの静かなトーン

## 4. Materials / Surface

- `素材感を合わせる`
  - 言い換え: シボ感、反射量、ハイライトの出方、エッジの硬さ、ムラ感を一致させる

- `マット`
  - 言い換え: 低光沢、反射弱め、落ち着いた表面

- `シボ`
  - 言い換え: レザー表面の細かな凹凸

- `スムースレザー`
  - 言い換え: なめらかで均一、シワや凹凸が少ない革表現

- `清潔感`
  - 言い換え: ほこりなし、汚れなし、擦れなし、白濁なし、整理された見え方

## 5. Background Replacement

- `背景を変える`
  - 推奨表現: `背景だけを差し替えた合成ではなく、シーン全体を一体で再撮影したように再生成する`

- `なじませる`
  - 言い換え: 同一レンズ、同一被写界深度、同一ノイズ粒状感、同一色温度、同一光源方向

- `浮いて見える`
  - 対策: 接地影、半影、微反射、輪郭ハロ禁止

## 6. Model / Wearing

- `モデル着用風`
  - 言い換え: 力の抜けた自然な姿勢、商品が主役の着用バランス、誇張しないポーズ

- `自然なポーズ`
  - 言い換え: 肩と首の力を抜く、片脚重心、手先を自然に整える

## 7. Useful Negative Terms

- `安っぽく見せない`
  - negative例: `cheap look, flat lighting, cheap laminate look, plastic-like surface`

- `別物化を防ぐ`
  - negative例: `bag redesign, altered proportion, changed hardware position`

- `背景差し替え感を防ぐ`
  - negative例: `cutout look, pasted subject, hard matte edge, halo edge`

## 8. Request Template

```text
目的:
- 何をどう見せたいか

維持:
- 形状
- 配置
- 構図
- 素材感

変更:
- 背景
- ライティング
- トーン
- ポーズ

質感指定:
- 反射量
- 陰影
- マット/光沢
- 清潔感

禁止:
- 別物化
- 黒つぶれ
- 白飛び
- 貼り込み感
```
